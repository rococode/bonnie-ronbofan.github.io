Enterprise Cloud Programs

Hybrid Multi-Cloud Workspace Platform

Citrix | Cloud Infrastructure

Led the program delivery for a major initiative to extend Citrix's workspace platform across hybrid multi-cloud environments. The program required coordinating infrastructure provisioning across AWS and Azure while maintaining seamless interoperability with on-premises deployments. Ronbo Fan managed cross-functional dependencies across six engineering teams, established release gating criteria, and delivered the capability on schedule with zero critical defects at GA.

DaaS Platform Scalability Program

Citrix | Desktop as a Service

Managed the end-to-end program for scaling Citrix's Desktop-as-a-Service platform to support significantly higher concurrent session counts for enterprise customers. The initiative involved capacity planning, performance benchmarking, database optimization, and load balancer tuning across multiple regions. Ronbo Fan coordinated performance engineering, SRE, and product teams to establish baseline metrics, identify bottlenecks, and deliver a 3x improvement in concurrent session capacity.

Enterprise Release Operations Overhaul

Citrix | DevOps & Process

Designed and implemented a comprehensive release operations framework for Citrix's cloud workspace division. The program replaced an ad-hoc release process with a structured, automated pipeline incorporating automated regression testing, staged canary deployments, feature flag management, and real-time rollback capabilities. This initiative reduced the average release cycle from weeks to days while improving deployment reliability.
